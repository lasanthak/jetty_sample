# jetty_sample

Sample Jetty (embedded) app.

Tested with Java 11.

Run: 
```mvn exec:java -Dexec.mainClass="org.lasantha.jetty.JettyApp"```

Sample queries:
* http://localhost:8080/lk/hello
* http://localhost:8080/lk/world
* http://localhost:8080/lk/echo/sklfjsdljfsdl?foo=bar&a=bb1&f
